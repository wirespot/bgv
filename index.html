Bicycle Geometry Visualiser

Visualise and compare bicycle geometries.
download .ZIPdownload .TGZ

Bicycle Geometry Visualiser (BGV) is a tool that allows you to define the geometry of one or more bicycles, see them rendered graphically on an HTML5 Canvas and perform visual comparisons by aligning several bicycles over each other. This can help you choose or compare frames and bicycle models.

For now BGV lacks a user interface and you will need to edit JavaScript files to make use of it. A user-friendly graphical interface is planned, but other features come first.

To use BGV, checkout the code and load index.html in a recent web browser (only tested with Firefox). Bicycle and parts definitions are in lib/definitions.js and you can control the rendering from lib/control.js.

Feedback is welcome. Please keep in mind that this is a very young project and the author's spare time is limited.

Feel free to hack on it and reuse the code (BGV is licensed under GPL v3), I am hoping that some of the graphical and trigonometrical libraries can be useful to others as well.
